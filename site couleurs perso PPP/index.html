Les différent code des couleurs

1.En nommant les couleurs en anglais.
Gris = Grey
Exemple: background-color: grey

2.En utilisant leur codes Hexadécimal.
Le code hexa de gris est: #808080
Exemple: background-color: #808080

3.En utilisant leur codes RGB.
Le code RGB de gris est:(128, 128, 128)
Exemple: background-color: rgb(128, 128, 128)

4.En utilisant leur codes HSL.
Le code HSL de gris est:(0, 0, 50%)
Exemple: background-color: hsl(0, 0, 50%)
